Loading

Please wait while the nook loads your items.

If this is the first time you're seeing this message after starting up, that means the app is starting wifi, logging in, downloading items, and setting everything up. Generally, this takes about 30 - 60 seconds

If you're seeing this after the first time, that means that the app couldn't load enough items fast enough for your viewing. Either you're loading a lot of items in a small amount of time or you have a slow internet connection.

Errors

If you receive an exception while logging in, that means you have an invalid username or password. Please correct your username & password in the '.settings/reader.settings' file and restart the app.

If you see any other exception at any time, please report it as a bug at http://code.google.com/p/nookapp-googlereader/issues/list
